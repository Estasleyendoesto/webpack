# Webpack base

Configuración inicial básica para cualquier tipo de proyecto estático estándar en webpack.

> Creado el 25-02-2021



#### Compatibilidad

```
sass, scss, css
imágenes
fuentes
csv
xml
toml
yaml
json5
```



## Uso

#### Instalar node primero

```
npm install
```



#### Development mode

```
npm run dev
```



#### Production mode

```
npm run build
```



#### Dev server

```
npm start
```
